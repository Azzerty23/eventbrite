# Eventbrite
### Project.THP by Thibault Mariolle & Augustin de Louvencourt

* Ruby version '2.5.1'
* Rails version '5.2.3'

Start:
- `$ rails db:create`
- `$ rails db:migrate`
- `$ rails db:seed`
- `$ rails server`

Go to: 
- `localhost:3000` <br>

- `https://eventbrite-thp-mtp.herokuapp.com/`
